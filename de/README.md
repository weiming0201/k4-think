# 德 — 退化到可操作的道

> 老子38章：失道而后德，失德而后仁，失仁而后义，失义而后礼。
> 每级退化丢掉不可形式化的部分，留下可操作的。= Shannon rate-distortion 的哲学版。

## 退化链

```
道（不可说）         → dao/wuji.md（空）
  ↓ 丢掉不可形式化的
德（可操作的道）     → dao/de/（本目录）
  ↓ 丢掉内容只留结构
k4（可操作的德）     → dao/k4/
  ↓ 丢掉方法只留规范
礼（可操作的输出）   → dao/de/li.md
```
