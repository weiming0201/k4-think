# K4 阴（反转）

探→拆→收→验。没有自己的看法时用。先听再想。

入口、门槛、核心规则见 `taiji.md`。

**大流程严格按序。不回退。** skill 内部的 4 步可以回退（黑盒）。𝔇 不够 → 跑新一轮，不是中间回退。

## 步间调度

### 探 → 拆

探完输出标准化结果。进拆（带着外部信息形成框架）。

### 拆 → 收

拆完输出标准化结果。进收。

### 收 → 验

收完输出标准化结果。进验（拿外部事实最终校验）。

### 验 → 结束或新轮

验完看 𝔇：
- 𝔇 ≥ 门槛 → 停
- Cauchy：|Δ𝔇| ≤ 0.05 连续两轮且 𝔇 ≥ 门槛 → 停
- 振荡 → 强制停，输出两方案
- 𝔇 < 门槛 → 新一轮（从探重新开始，带着本轮结果作为 context）
- 轮数 ≥ 5 → 强制停

## 逆链追问

每步做完追问一句。不改 𝔇。

| 做完 | 追问 |
|------|------|
| 探 | 删掉搜到的一切，后验变不变？（Bayesian surprise） |
| 拆 | 给不了解P的人看四个维度，能理解吗？ |
| 收 | 这个结论K4之前就隐约知道吗？ |
| 验 | 选3个●，反面论据推翻得了吗？ |
