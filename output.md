# K4 输出翻译

K4 跑完产出的是结构化产物（维度、矩阵、𝔇、逆链标记）。给不同受众看需要不同翻译。

## 翻译原则

1. **结论先行**。一句话结论放最前面。不是"经过四步分析我们发现"——是"结论是X"
2. **不提 K4 术语**。没有 γπαυ，没有 S₂，没有 𝔇。读者不需要知道你怎么想的，只需要知道你想到了什么
3. **保留发现不保留过程**。拆了4个维度 → 翻译成"这个问题有4个关键面"。验了16格 → 翻译成"我们检查了X个方面，Y个确认、Z个存疑"
4. **不确定性要说人话**。𝔇=0.85 → "我们对这个结论比较有信心，但有几个点还需要确认"。○格子 → "有X个方面我们目前没有答案"

## 逐步翻译

### 拆的产出 → 人话

K4 产出：四个维度 A/B/C/D + S₂配对 + 𝔇

翻译模板：
```
这个问题有四个关键面：
1. [A 的人话名] — [一句话说这个面关注什么]
2. [B 的人话名] — [一句话]
3. [C 的人话名] — [一句话]
4. [D 的人话名] — [一句话]

其中 [A和B] 是一对（都关注[共同特征]），[C和D] 是另一对。
这四个面覆盖了问题的主要方面。[如果CE反查发现了边界：但没有覆盖[X]，这个需要单独注意。]
```

**不翻译的**：S₂ 的具体配对表、K3 退化测试结果、𝔑 分数。这些是过程，不是结论。

### 探的产出 → 人话

K4 产出：四源搜索结果 + 三件套检验

翻译模板：
```
我们从四个角度调研了这个问题：
- 学术研究怎么说：[核心发现]
- 工程实践怎么做：[核心发现]
- 做过这件事的人怎么看：[核心发现]
- 其他领域有没有类似问题：[核心发现]

主要发现：[最重要的1-3条，用陈述句]
意外发现：[如果有框架冲击，翻译成"我们原来以为X，但调研发现Y"]
信息缺口：[如果某源为空，翻译成"关于X方面目前公开信息很少"]
```

### 验的产出 → 人话

K4 产出：16格吻合度 + ●/○ + 发展度矩阵

翻译模板：
```
我们逐项检查了 [问题的四个面] × [四个分析角度]，共 [N] 项：
- [X] 项有明确的事实支撑
- [Y] 项存疑（有相关信息但不完全吻合）
- [Z] 项目前没有答案

主要盲点：[○格子翻译成"我们不知道[具体什么]"]
主要风险：[如果有致命盲点："如果[X]的答案和我们假设的不同，整个结论可能需要调整"]
```

**不翻译的**：4×4 矩阵本身、发展度数值、行均列均。这些是内部工具。

如果对方是技术受众，可以附上矩阵作为附录，但正文用人话。

### 收的产出 → 人话

K4 产出：矩阵规律 + 矛盾 + PI + 压缩结论 + 𝔇

翻译模板：
```
结论：[一句话压缩结论]

支撑：
- [从验的●翻译的关键事实，2-3条]

风险：
- [从矛盾/○翻译的不确定性，1-2条]

下一步：
- [如果𝔇够：建议执行X]
- [如果𝔇不够：建议先确认Y再决定]
```

**不翻译的**：PI 的思想实验过程、Cauchy 停止判断、矩阵形状名称。

## 置信度翻译

| 𝔇 范围 | 人话 |
|---|---|
| ≥ 0.9 | 我们对这个结论很有信心 |
| 0.8-0.9 | 我们比较有信心，但有几个点需要确认 |
| 0.7-0.8 | 初步判断是X，但还需要更多信息 |
| < 0.7 | 目前信息不足以下结论，建议[具体需要什么] |

---

## 盲点翻译

| K4 产出 | 人话 |
|---|---|
| ○ 格子 | 关于[X]我们目前没有答案 |
| 致命盲点（○且会改变结论） | 如果[X]的情况和我们假设的不同，结论可能翻转 |
| 流浪事实 | 有一些信息不属于我们的分析框架，可能意味着我们的分析角度不够全 |
| ◐ 偏移 | 关于[X]有一些信息但和我们预期的不完全一致 |
| PI 失败 | 换个角度分析可能得到不同结论 |
| 矛盾 | [A]和[B]的信息互相矛盾，需要进一步确认 |

## 反模式

❌ **K4 术语外泄**："经过 S₂ 配对检验和 K3 退化测试"——读者不知道你在说什么
❌ **过程当结论**："我们拆了四个维度，探了四个源，验了16格"——读者要的是结果不是过程
❌ **数字堆砌**："𝔇=0.87，覆盖率3/4，正交度5/6"——翻译成人话再说
❌ **假精确**："我们有87%的信心"——𝔇 不是概率。说"比较有信心"
❌ **隐藏不确定性**：有○不说，有矛盾不提。不确定性是结论的一部分
