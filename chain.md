# K4 阳（正转）

拆→探→验→收。有自己的判断时用。默认模式。

入口、门槛、核心规则见 `taiji.md`。

**大流程严格按序。不回退。** skill 内部的 4 步可以回退（黑盒）。𝔇 不够 → 跑新一轮，不是中间回退。

## 步间调度

### 拆 → 探

拆完输出标准化结果。不管 𝔇 多少都进探。
𝔇 ≥ 门槛 → 标记可提前停（但正环默认走完）。

### 探 → 验

探完输出标准化结果。进验。

### 验 → 收

验完输出标准化结果。进收。

### 收 → 结束或新轮

收完看 𝔇：
- 𝔇 ≥ 门槛 → 停
- Cauchy：|Δ𝔇| ≤ 0.05 连续两轮且 𝔇 ≥ 门槛 → 停
- 振荡 → 强制停，输出两方案
- 𝔇 < 门槛 → 新一轮（从拆重新开始，带着本轮结果作为 context）
- 轮数 ≥ 5 → 强制停

## 逆链追问

每步做完追问一句。不改 𝔇。

| 做完 | 追问 |
|------|------|
| 拆 | 给不了解P的人看四个维度，能理解吗？ |
| 探 | 删掉搜到的一切，后验变不变？（Bayesian surprise） |
| 验 | 选3个●，反面论据推翻得了吗？ |
| 收 | 这个结论K4之前就隐约知道吗？ |
