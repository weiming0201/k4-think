# K4 参考手册

> 理论基础：`skills/k4-think/references/k4_meta.md`
> 触发：BOOTSTRAP.md §2（数不可逆信号）
> 路由：每个子skill输出末尾的路由指令
> 状态：`memory/k4_state.md`

## 碱基功能表

|  | 看γ | 排π | 比α | 问υ |
|---|---|---|---|---|
| **γ存在** | 拆1:回看已有 | 拆2:K4拆分 | 拆3:验证正交 | 拆4:判断置信 |
| **π结构** | 验2:搜索打分 | 验1:搭建框架 | 验4:找最大缺口 | 验3:验证合理 |
| **α区分** | 探3:汇总可能 | 探4:组织结构 | 探1:辨别方向 | 探2:写搜索prompt |
| **υ自指** | 收4:确认行动 | 收3:调整结构 | 收2:调整方向 | 收1:元质疑问题 |

## 旋转序列

```
拆：γ→π→α→υ (看→排→比→问)  恒等置换 (1,2,3,4)
探：α→υ→γ→π (比→问→看→排)  隔位交换 (3,4,1,2)
验：π→γ→υ→α (排→看→问→比)  邻对交换 (2,1,4,3)
收：υ→α→π→γ (问→比→排→看)  完全逆序 (4,3,2,1)
```

## 执行协议

1. **入口**：BOOTSTRAP §2 决定是否用 K4 及深度
2. **首步**：永远从拆开始。读 decompose SKILL.md
3. **路由**：每步输出末尾的路由指令决定下一步。agent 直接执行
4. **状态**：每步写 `memory/k4_state.md`，下步从此文件读上下文
5. **约束**：思行分离（K4完成前不执行外部动作）；轮数≤3

## 自适应深度

不预选模式。拆完看 𝔇 决定继续还是停：

```
拆 → 𝔇≥0.8 停 → 事后看：这是"快速模式"
拆 → 𝔇<0.8 → 探 → 停 → 事后看：这是"标准模式"
拆 → 探 → 验 → 收 → 停 → 事后看：这是"完整模式"
```

模式是轨迹的事后描述，不是预先选择。
