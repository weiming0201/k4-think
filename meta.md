## 1. 四碱基（原子操作）

| 碱基 | 符号 | 编号 | 做什么 | 对应维度 |
|---|---|---|---|---|
| **看** | γ | 1 | 获取信息：观察存在的东西 | 存在 |

## 3. 四条路径

### 3.1 语义

```
        看问题          校验
内部     拆              收
外部     探              验
```

| 路径 | 旋转 | 认知定位 | 做什么 |
|---|---|---|---|
| **拆** | γ→π→α→υ | known knowns | 我怎么看这个问题。拆成四个正交维度 |
| **探** | α→υ→γ→π | known unknowns | 别人怎么看。四信息源搜索 |
| **验** | π→γ→υ→α | unknown unknowns | 外部事实和框架对得上吗。涌现层 |
| **收** | υ→α→π→γ | making it coherent | 内部自洽吗。矩阵规律+矛盾+压缩 |

### 3.2 Klein 四元群

四个置换构成 V₄：

| 置换 | 路径 | 群元素 |
|---|---|---|
| (1,2,3,4) | 拆 | e（恒等） |
| (2,1,4,3) | 验 | a |
| (3,4,1,2) | 探 | b |
| (4,3,2,1) | 收 | c |

群运算表：

|  | e(拆) | a(验) | b(探) | c(收) |
|---|---|---|---|---|
| **e(拆)** | 拆 | 验 | 探 | 收 |
| **a(验)** | 验 | 拆 | 收 | 探 |
| **b(探)** | 探 | 收 | 拆 | 验 |
| **c(收)** | 收 | 探 | 验 | 拆 |

性质：封闭（任意两个组合=第三个）、自逆（每个执行两次=恒等）、交换（顺序无关）。
四条路径不是任意选择——是唯一满足群封闭性的四个排列。

### 3.3 镜像对

```
拆 (1234) ←逆序→ 收 (4321)    纵向环：不搜索，内生认知
探 (3412) ←逆序→ 验 (2143)    横向环：外部搜索
```

### 3.4 耦合

同环自然续接（终止维度=起始维度）：
```
拆(终止υ) → 收(起始υ)
收(终止γ) → 拆(起始γ)
探(终止π) → 验(起始π)
验(终止α) → 探(起始α)
```

跨环连接：
```
拆(纵向) → 探(横向)    拆的维度 → 探的搜索方向
验(横向) → 收(纵向)    验的矩阵 → 收的分析对象
```

## 4. 调度（正链 chain.md）

6 种固定路径，大流程不回退。三种旋转：

```
正转（阳，默认）：拆→探→验→收  有自己的判断时
反转（阴）：    探→拆→收→验  没有自己的看法时，先听再想
速查（对角线）： 拆→验 / 探→收  时间紧时的快速互反对撞
```

```
问题P到来 → 数不可逆信号 → 选旋转 → 严格按序走完
𝔇门槛 = 0.8 + 0.05 × 不可逆信号数（上限0.95）— Neyman-Pearson
一轮走完 𝔇 不够 → 新一轮（带本轮结果重开）
终止：𝔇 ≥ 门槛 / Cauchy / 振荡 / 5 轮
```

核心规则：
1. 大流程不回退。skill 内部可回退（黑盒）
2. 𝔇 是轮末出口门
3. 停是默认
4. 单人完整回路
5. 每步强制写下来。不写 = 没做
6. 外部模型视角是工具不是协作

## 5. 逆链（antichain.md）

DNA 双螺旋：正链建设，逆链解构。每步完成后一句话追问，不改 𝔇，做元认知标记。

| 步骤 | 逆链追问 |
|---|---|
| 拆 | 给不了解P的人看这四个维度，他能理解吗？ |
| 探 | 删掉搜到的所有东西，后验变不变？（Bayesian surprise） |
| 验 | 选3个●格子，试着用反面论据推翻它 |
| 收 | 这个结论K4之前我就隐约知道吗？ |

## 6. 各步核心机制

### 6.1 拆 (decompose.md)

把问题拆成四个正交维度。

**检验**：
- S₂ 配对（Yoneda + Shannon 互信息 I=0）：维度由关系定义，正交=无共享信息
- K3 退化（维度独立性）：去掉一个维度后秩下降
- 𝔑 非显然性：在已知分类法中找不到相同维度
- CE 反查（FMEA）：穷举死法检验覆盖
- Noether 第二定理：gauge 约束检查——维度间有没有锁定关系

**存在性保证**（Noether 正规化）：有限复杂度问题可用有限维度展开。4 个够。

### 6.2 探 (explore.md)

四信息源固定搜索，不随问题变。

| 信息源 | K4 | 看什么 | 质量标准 |
|---|---|---|---|
| 学术界 | α | 概念边界、taxonomy | 同行审议、引用网络、优先 primary |
| 工业界 | π | 代码、架构、benchmark | 真跑过、有代码、可复现 |
| 亲历者 | γ | 踩坑、决策理由、复盘 | 本人做过、有具体细节 |
| 平行领域 | υ | 同构问题的解法 | 不同学科、结构同态 |

**存在性保证**（Shannon 噪声信道编码）：信噪比够则信号可提取。三件套测信道够不够。
**信道容量**（Shannon）：同一源搜 10 次不如 4 源各搜 2 次。

**硬检验**：源覆盖率（≤2/4 补搜）、信息正交性（正交基）、框架冲击度（Bayesian update）。

### 6.3 验 (validate.md)

16 格必填。K4 意义完备——填不上是盲点不是"没意义"。

**前提**（Gödel 完备性）：意义完备 → 真命题可证 → 格子空 = 不完备。
**极限**（Gödel 不完备性）：完备 + 一致不可兼得。矛盾不是 bug 是系统极限。

**两层打分**：
1. ●/○：能不能填（unknown unknowns 检测）
2. 0-1 发展度：填到什么程度（瓶颈检测）

**事实确定性**：多源交叉=确定、单源 primary=条件确定、单源 secondary=待验证、观点=不是事实。

**硬检验**：●全检（Type I error，假阳性比○更危险）、○深挖、流浪事实、外部出口。

产出发展度矩阵，传给收做规律分析。验只产出原始数据。

### 6.4 收 (converge.md)

内部一致性。不搜外部，纯内生。

**矩阵规律**（Noether 同调群：从数字到结构）：
读验的发展度矩阵——行均/列均/对角线/梯度/孤立低分/整片缺口。
形状本身是诊断。

**矛盾检测**：
维度间矛盾、事实间矛盾、层间一致性（拆/探/验 说的一致吗）。
矛盾 = 1 可能是 Gödel 不完备性的代价，不一定是拆的错误。

**Procedure Invariance**（Noether 第一定理）：
换条路走到同一终点 = gauge invariant。不一样时两种可能：路径产物（标注，新一轮重审）或多不动点（Knaster-Tarski，输出两方案）。

**压缩展开**（Shannon DPI）：
一句话结论 → 能无损展开回完整过程吗？展不开 = 丢了信息。

**收敛判据**：
Banach/Cauchy：|Δ𝔇| ≤ 0.05 连续两轮且 𝔇 ≥ 门槛 → 停。
病态：Δ𝔇 < 0 连续两轮 → 强制停。振荡 → 输出两方案。轮数 ≥ 5 → 强制停。

## 8. K3 退化与生长

### 退化

| 缺少 | 退化 |
|---|---|
| 拆 | 搜索方向不知从何而来 |
| 探 | 闭门造车 |
| 验 | 盲区不自知 |
| 收 | 永动讨论机 |

### K3 的生长性

K4 是对称态（稳定但静止）。K3 是不对称态（有方向、有动力）。

```
K4 结晶 → 发现内在矛盾 → 悬置矛盾最大的维度 → K3
K3 自由发展 → 产出新结构 → 补完第四维 → K4'
K4' ≠ K4 → 新一轮
```

不传内容，传缺口。K3 的缺失维度 = 传给下一轮的唯一信息 = 生长方向。

## 11. 公理

1. **可孤立** — 每个元素独立自洽（V₄ = Z₂×Z₂ 直积结构）
2. **可配对** — S₂ 三配对都有非平凡意义（V₄ 恰有 3 个 Z₂ 子群）
3. **来自三** — 上层 K3 生成（K3 退化验证必要性）
4. **生成三** — 三元组生成下层（K3 是下一轮的种子）
5. **四则完备** — 缺一不成（V₄ 是最小非循环群）

S₂ 严格化：命名测试、反转测试、稳健性（换分组依据不变）。

K4 借用 V₄ 的部分性质，但 K4 ≠ V₄。K4 额外包含层间生成、补完方向、可孤立性。K4 是 V₄ 的语义推广。

## 12. 跨文化四步范式映射

K4 的拆探验收不是孤立发明。多个文明独立产生了四步推理结构。

| 范式 | 来源 | 四步 | K4 映射 |
|---|---|---|---|
| **起承转合** | 中国文论 | 起（提出）→承（展开）→转（转折）→合（收束） | 拆→探→验→收 |
| **因明四句** | 印度因明学 | 宗（命题）→因（理由）→喻（类比）→合（结论） | 拆→探→验→收 |
| **正反合** | 黑格尔辩证法 | 正题→反题→合题→新正题 | 拆→逆链→收→新一轮 |
| **四句否定** | 佛教 Catuṣkoṭi | 是→非→亦是亦非→非是非非 | 拆→逆链→验→收（超越二值） |
| **四因** | 亚里士多德 | 质料因→形式因→动力因→目的因 | γ→π→α→υ（直接映射四维度） |
| **OODA** | 军事决策 Boyd | Observe→Orient→Decide→Act | 看→拆→收→行动（含执行） |
| **PDCA** | 质量管理 Deming | Plan→Do→Check→Act | 拆→执行→验→修正（含执行） |
| **科学方法** | 近代科学 | 假说→实验→观察→结论 | 拆→探→验→收 |
| **诠释学循环** | Gadamer | 前理解→文本→理解→新前理解 | 拆→探→收→新一轮 |

**模式**：
- 起承转合、因明四句、科学方法和 K4 几乎 1:1
- 亚里士多德四因直接映射 γπαυ（质料=存在、形式=结构、动力=区分、目的=意义）
- 正反合和 Catuṣkoṭi 映射正链+逆链的对撞结构
- OODA/PDCA 包含行动步骤——K4 产出结论不产出行动，行动在 K4 之后

**独立收敛**：不同文明不同时代产生了结构相似的四步范式。这不是 K4 的来源——是 K4 意义完备性的跨文化佐证。
