# K4 太极（入口 + 门槛 + 规则）

## 入口

```
问题P到来
  ↓
数不可逆信号（行动不可逆 / 误读代价致命 / 直觉无用 / 后果影响他人）
  0 → 不用K4。直觉就行
  1 → 分。𝔇≥0.8可停
  2+ → 分→筛→搜→归，𝔇≥0.9才停
```

空输入/模糊问题：先用分的 Step 1（看×γ：观察已有事实）理清 P 是什么。

## 𝔇 门槛

```
𝔇门槛 = 0.8 + 0.05 × 不可逆信号数（上限0.95）— Neyman-Pearson
  信号1 → 0.85
  信号2 → 0.90
  信号3+ → 0.95
```

## 四象速查

```
        扩张    收缩
内部     分      归
外部     搜      筛
```

对角线：分↔筛（AT配对），搜↔归（CG配对）。AT对完成才放CG对。

| 碱基 | 标准化输出 |
|---|---|
| **分** | 四维度 + S₂/2×2/K3/𝔑/CE + 𝔇 |
| **搜** | 四源结果 + 三件套(覆盖率/正交性/冲击度) |
| **筛** | 16格吻合度 + 发展度矩阵 + ○深挖 + ●全检 |
| **归** | 矩阵形状 + 矛盾 + PI + 压缩结论 + 𝔇 |

## 核心规则

1. **配对强制。** 分→筛→搜→归，不跳对
2. **𝔇 是出口门。** 一轮走完看 𝔇 决定停还是新一轮
3. **停是默认。** 大多数问题分完就够
4. **每步强制写下来。** 不写 = 没做
5. **不回退。** 𝔇不够 → 新一轮（带本轮结果重开），不回退

## 反模式

❌ 信号=0时强行启动K4
❌ 跳过配对：分完直接搜
❌ 不写就说做完了
❌ 为了提高𝔇反复磨同一轮
